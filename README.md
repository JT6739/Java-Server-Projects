# Java Server Projects
This repository contains Java networking programs that demonstrate client-server communication using both UDP and TCP sockets. The projects were built to practice core networking concepts such as packet-based communication, reliable stream-based communication, HTTP request/response handling, round-trip time measurement, and multithreaded server design. 
This was done using two Linux Server machines (one for server, the other for client) by using powershell to upload the files on to each machine and test each project file

## UDP-Server-Project
Java UDP client-server inventory lookup system. The client sends an item ID to the server using UDP sockets, receives product information, and calculates round-trip time.
Key concepts learned:
- UDP sockets
- Datagram packets
- Client-server communication
- Item lookup system
- Round-trip time measurement

## simple-http-server
Multithreaded Java HTTP server and client built with TCP sockets. Supports GET requests, basic HTTP response codes, file transfer, and threaded client handling.
Keyconcepts learned:
- TCP sockets
- HTTP GET requests
- HTTP response status codes
- File transfer
- Multithreaded server design
- Round-trip time measurement
